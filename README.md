# Student Exam Management System

Console-based Java application using OOP principles.

## Features
- Add students
- Add exams to students
- Calculate average scores
- Determine pass/fail status
- Find top student

## Business Rules
- Exam score range: 0–100
- Exam is passed if score ≥ 50
- Student passes if:
  - Average score ≥ 60
  - All exams are passed
- Student with no exams automatically FAILS
- Top student must have at least one exam

## Technologies
- Java
- OOP
- ArrayList
- Console Input (Scanner)

## How to Run
1. Compile all `.java` files
2. Run `Main` class
